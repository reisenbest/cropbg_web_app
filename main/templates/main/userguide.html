Требования к изображениям:

  • Фон. Фон должен контрастировать с артефактом и масштабом. Фон должен быть однотонным.
  • Артефакт. Артефакт должен четко выделяться на фоне, быть в фокусе.
  • Масштаб. Масштаб не должен быть нарисован на фоне. Масштаб не должен быть прозрачным. Масштаб должен контрастировать с фоном (в идеале - цветная однотонная непрозрачная линейка).
  • Освещение. Нужно хорошее ровное освещение без перепадов. Постарайтесь минимизировать тени на границах артефакта и фона.
  • Профили. Для фотографий профилей также необходим контраст между артефактом, подставкой для артефакта и фоном. Однако, скорее всего придется отдельно вырезать подставку:) В нашем мире нет ничего идеального.

Цвет фона:

При загрузке изображения вы можете выбрать цвет фона. Вот некоторые пояснения

Цвет фона нужно задать в формате RGBA где:
  • R - отвечает за красный, значение может быть в пределах от 0 до 255.
  • G - отвечает за за зеленый, значение может быть в пределах от 0 до 255.
  • B - отвечает за за синий, значение может быть в пределах от 0 до 255.
  • A - отвечает за прозрачность фона, значение может быть в пределах от 0 до 255, где 0 - совсем прозрачный, 255 - нулевая прозрачность.
В форме нужно ввести фон в формет 4 цифр, разделенных запятыми, не более 255, без пробелов. Вот пример:
  • 0,255,0,255 - зеленый непрозрачный фон
  • 255,0,0,128 - красный полупрозрачный фон
  • И так далее
Цвет можно подобрать в любом колор-пикере онлайн.
